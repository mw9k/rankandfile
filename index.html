About

  • Chess beginners may struggle to name the coordinates of a given square. This simple web game is designed to help train that ability.
  • The site was designed especially for mobile. Tapping the multiple choice options is easiest on a touchscreen device.
  • Keyboard Shortcut: use A/S/D to make a multiple choice selection.
  • Recommended strategy: start in File Only or Rank Only mode. Consider using the "Show Quadrants" aid. As you progress, try File & Rank mode, with fewer pieces, and even flipped.
  • The multiple choice options are generated deliberately close to the correct answer. Hopefully this will prevent reliance on multiple choice guessing strategies.
  • Settings are saved using localStorage; similar to cookies. Only the settings are saved; there is no tracking etc.
  • Requires a modern browser.
